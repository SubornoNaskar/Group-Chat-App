# Group Chat App

A real-time group chat backend built with:

- Spring Boot
- WebSocket
- STOMP
- MongoDB

## Features

- Create chat rooms
- Join chat rooms
- Send real-time messages
- Store chat history in MongoDB

## Tech Stack

Backend:
- Java
- Spring Boot
- WebSocket/STOMP
- MongoDB
